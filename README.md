## How To Open

1. 克隆或下载本仓库。
2. 安装并打开嘉立创EDA / EasyEDA。
3. 在对应项目目录中打开 `.eprj2` 工程文件。

## Notes

- 本仓库用于展示硬件工程设计能力，包含原理图与 PCB 工程源文件。
